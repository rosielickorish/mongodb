# Building Query Documents
Run this application by typing:

node buildingQueryDocuments.js

This application runs with the following options passed in to build three different queries:

var allOptions = [
    {
        firstYear: 2002,
        lastYear: 2016,
        city: "Palo Alto"
    },
    {
        lastYear: 2010,
        city: "New York"
    },
    {
        city: "London"
    }
];
